Diferença entre IF e ELSE

If

Executa um bloco de código somente se a condição for verdadeira.

Else

Executa um bloco de código quando a condição do if for falsa.

Eemplo

    let idade = 17;
    if (idade >= 18) {
    console.log("Pode dirigir");
    } else {
    console.log("Não pode dirigir");
    }
