⚡ Exercícios e Projetos em JavaScript

Este repositório reúne exercícios e projetos desenvolvidos com JavaScript, com foco no fortalecimento da lógica de programação e na criação de sites interativos.

Ao longo do conteúdo, foram abordados conceitos essenciais como variáveis, condicionais, estruturas de repetição, funções e manipulação do DOM, além da aplicação prática na construção de interfaces dinâmicas.

O projeto também inclui o uso do Bootstrap para criação de layouts responsivos e componentes visuais, integrando JavaScript ao desenvolvimento de páginas web mais completas e funcionais.

Os exercícios e projetos demonstram a evolução no desenvolvimento front-end, unindo lógica, interatividade e design em aplicações reais.

🚀 Tecnologias utilizadas: JavaScript, HTML5, CSS3 e Bootstrap
📚 Foco: Lógica de programação, interatividade, responsividade e interfaces modernas
💡 Objetivo: Consolidar conhecimentos em JavaScript e desenvolver aplicações web completas
